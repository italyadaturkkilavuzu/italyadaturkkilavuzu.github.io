    Burslar ve Finansal Destek #

    DSU (Diritto allo Studio Universitario) Bursları #

    Başvuru Koşulları #

    • Gelir testi (ISEE)
    • Akademik başarı kriterleri
    • İtalya’da ikamet koşulları

    Başvuru Süreci #

    1. ISEE belgesinin alınması
    2. Online başvuru formu
    3. Gerekli belgelerin yüklenmesi
    4. Sonuç takibi

    Burs Miktarları #

    • Minimum: 1.500€/yıl
    • Maksimum: 5.000€/yıl
    • Barınma desteği: 2.000-4.000€/yıl
